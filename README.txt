These scripts are written to be compatible with the first release of the Unity Editor VR Experimental build. The build is under development, so no guarantees can be made as to whether they will still be compatible even in the near future.

To install and configure Editor VR, follow the instructions here: https://docs.google.com/document/d/1IFQve5gAOb1gQzIhhtEr3WLrctJhsoJxl6j07pg2DYA/edit

To install this HierarchyMode tool, copy and paste all files in this repository into a subfolder named HierarchyMode in Assets/EditorVR/Tools/
